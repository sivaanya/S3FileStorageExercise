Spring Boot Project to upload and delete file to/from AWS S3 bucket:

What: Spring Boot project for uploading and deleting files to/from AWS S3 bucket
using Basic static credential method.

why: To get hands-on in AWS S3 with Springboot.

Pre-requisite:
a. Please create a bucket(springbootstorage) in S3 using AWS console before running this project.
b. Get access & secret from Aws console to update them in YAML file.
